# 1:1 메시지(쿠폰/개인 알림) 수동 수집 — 구글폼 설계

## 왜 수동인가
카카오톡 개인 1:1 채팅 메시지는 카카오 공식 API로 외부 수집이 불가능하고,
비공식 자동화는 약관 위반·계정정지 리스크가 있어 자동화하지 않습니다.
대신 입력 시간을 30초 이내로 줄이는 폼 구조로 운영 부담을 최소화합니다.

## 사전 준비
4개 채널(르무통/아디다스/나이키/스케쳐스)을 모니터링용 휴대폰(또는 개인 계정)에서
플러스친구 추가 → 알림 수신 ON

## 구글폼 항목 구성

1. **브랜드** (객관식, 필수)
   - 르무통 / 아디다스 / 나이키 / 스케쳐스

2. **수신 시각** (날짜+시간, 필수)
   - 기본값: 폼 제출 시각이 자동 기록되므로 별도 입력 없이 타임스탬프 활용 가능
   - 단, 메시지를 "받은 시각"과 "입력한 시각"이 다를 수 있어 별도 시간 입력란 추천

3. **메시지유형** (객관식, 필수)
   - 쿠폰/할인 · 신상품 출시 · 시즌세일 · 이벤트/응모 · 단순공지 · 기타

4. **메시지내용** (단답형 또는 장문형, 필수)
   - 텍스트 그대로 붙여넣기

5. **이미지 첨부** (파일 업로드, 선택)
   - 메시지 내 이미지/배너 스크린샷

6. **링크/버튼 URL** (단답형, 선택)
   - 메시지 내 "구매하기" 등 버튼이 연결된 URL

7. **메모** (장문형, 선택)
   - 벤치마킹 포인트, 자사 대비 차이점 등

## 응답 시트 연동
- 폼 생성 시 "응답 → 스프레드시트 보기"로 자동 연동되며,
  A안 크롤러가 쓰는 동일한 스프레드시트(`카카오채널_모니터링`)에
  새 시트(`1on1_메시지`)로 추가하면 한 파일에서 공개소식/1:1메시지를 함께 관리 가능

## 통계 자동화
응답 시트에 아래 함수를 넣으면 자동 집계됩니다.

```
=QUERY('1on1_메시지'!A:G, "select A, count(A) where A is not null group by A label count(A) '메시지건수'")
```

→ 브랜드별 누적 발송건수가 자동으로 갱신되는 통계 영역 생성

## 운영 팁
- 알림 받는 즉시 입력하는 습관이 핵심 (모아서 하면 누락 발생)
- 메시지 빈도가 높은 시즌(블프/시즌오프)에는 주 1회 일괄 정리 타임 확보 권장
